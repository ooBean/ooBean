### 🎨 腾讯设计云 (Tencent Codesign)
**[🌐 在线演示](https://codesign.qq.com/)**

**📋 项目性质 | Project Nature**
- **企业级设计协作平台** - 为腾讯产品设计团队提供一站式协作工具
- **大型复杂系统** - 支持设计标注、切图生成、图标库调用、数据埋点等功能

**🎯 核心功能 | Core Features**
- 🎨 **设计标注**: 自动标注设计稿尺寸、颜色、字体等信息
- ✂️ **切图生成**: 一键生成不同分辨率的切图资源
- 🖼️ **图标库**: 集成腾讯内部图标库，支持快速调用
- 📊 **数据埋点**: 用户行为数据采集和分析

**🏗️ 技术实现 | Technical Implementation**
- **前端框架**: Vue 2 + Vuex
- **网络请求**: Axios + Promise
- **状态管理**: Vuex + 模块化状态管理
- **性能优化**: 虚拟滚动、懒加载、代码分割

**💡 项目亮点 | Project Highlights**
- 🌐 **大型平台**: 服务腾讯生态的设计协作平台，用户量级大
- ☁️ **云端协作**: 设计源文件云端备份，版本历史回溯
- 💬 **评论系统**
> 实现了在设计稿任意位置添加评论的功能，系统会记录该位置的XY坐标和评论内容，重新打开时精准还原到原位置，大大提高了设计评审效率。
- 🔐 **权限管理**
> 实现了完善的团队协作权限控制系统，支持不同角色对设计稿的不同操作权限，确保了设计稿的安全性和协作的规范性。
- 📝 **便签功能**
> 实现了支持富文本编辑的便签系统，用户可以在设计稿上添加便签，支持文本框输入、换行、移动、调整大小等操作，提高了团队协作效率。
